# -Array-Operations-using-Class-in-C-
Design a C++ program using Object-Oriented Programming concepts to perform basic operations on an integer array.

Create a class named ArrayOperations that contains data members and member functions to
perform the following tasks:
1. Accept n integer elements from the user.
2. Display the array elements.
3. Find and display the maximum and minimum elements.
4. Calculate and display the sum and average of elements.
5. Search for a given element in the array and display its position (if found).
